# ARTVHR DEV

Сайт-портфолио веб-разработчика. Хостится на **GitHub Pages**.

## Файлы

```
├── index.html           # Главная страница
├── maxweqq-cover.png    # Обложка проекта Maxweqq (также используется как og:image)
└── README.md
```

## Деплой на GitHub Pages

1. Залей все файлы в репозиторий на GitHub
2. Settings → Pages → Source: **Deploy from a branch** → `main` / `root`
3. Сайт будет доступен по адресу `https://ИМЯ.github.io/РЕПО/`

## Что заменить в index.html

| Что | Сколько мест |
|-----|-------------|
| `YOUR_GITHUB_USERNAME` | 2 — в og:url и og:image |
| `YOUR_REPO` | 2 — в og:url и og:image |
| `YOUR_TG` | 3 — кнопки и навбар |
| `YOUR_IG` | 1 — кнопка Instagram |
| `XXXXXXXX` | 2 — номер счётчика Yandex.Metrica |
